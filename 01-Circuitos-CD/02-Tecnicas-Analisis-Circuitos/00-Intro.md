# Módulo 2: Técnicas para el Análisis de Circuitos

## Descripción
Este módulo presenta las técnicas sistemáticas para analizar circuitos eléctricos complejos utilizando métodos de nodos y mallas.

## Objetivos de Aprendizaje
Al finalizar este módulo, el estudiante será capaz de:
- Comprender la topología de redes eléctricas
- Aplicar el método de nodos para resolver circuitos
- Aplicar el método de mallas y lazos

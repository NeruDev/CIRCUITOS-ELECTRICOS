# Módulo 5: Análisis de Circuitos de Segundo Orden

## Descripción
Este módulo analiza circuitos que contienen dos elementos almacenadores de energía, cuya respuesta se describe mediante ecuaciones diferenciales de segundo orden.

## Objetivos de Aprendizaje
Al finalizar este módulo, el estudiante será capaz de:
- Formular ecuaciones diferenciales de segundo orden
- Identificar los tipos de respuesta (sobreamortiguada, críticamente amortiguada, subamortiguada)
- Resolver circuitos RLC serie y paralelo

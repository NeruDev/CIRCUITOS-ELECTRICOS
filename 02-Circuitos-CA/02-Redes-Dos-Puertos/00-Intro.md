# Módulo 2: Redes de Dos Puertos

## Descripción
Este módulo presenta el análisis de redes de dos puertos utilizando diferentes conjuntos de parámetros.

## Objetivos de Aprendizaje
Al finalizar este módulo, el estudiante será capaz de:
- Definir y calcular parámetros de redes de dos puertos
- Convertir entre diferentes tipos de parámetros
- Analizar interconexiones de redes
